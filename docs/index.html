CRUD

Sqlite3-attend mainly focus Create, Read, Update and Delete operations on sql database.

JSON Objects

From creating database to running CURD operations, everything is done using JSON objects.

Asynchronous Wrapper

Sqlite3-attend is just a wrapper around sqlite3 library. All the operations are asynchronous by default.
